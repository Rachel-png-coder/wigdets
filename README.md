# the divider widget 
- A Divider is a visual element used to separate content into sections.

- It improves readability, structure, and overall user interface design.
- Divider – for horizontal separation,, (top and bottom). Its horizontal by default.
- Mostly used in Column or ListView, height: controls the total vertical space the divider occupies
- Used to properly arrange side-by-side content, we usually combine Row with an expanded widget, which allows both - columns to share the available screen space evenly.
- We also have the VerticalDivider – for vertical separation 
- VerticalDivider is used to create a clear separation between the left and right sections
